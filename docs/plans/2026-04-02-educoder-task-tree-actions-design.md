# Educoder 任务树当前态与右键菜单设计文档

**日期：** 2026-04-02  
**目标：** 给 Task Tree 补上“当前题目”标记，并在题目节点上提供常用右键动作，减少频繁切到 Workbench 或命令面板的操作成本。

## 1. 范围

本轮只做两个增强：

1. 当前已打开题目在树中显示为“当前”
2. 题目节点右键菜单提供：
   - Sync Answers
   - Sync Full Repository
   - Compare With Template
   - Compare With Answer

## 2. 方案

- **当前题标记**：不做复杂 TreeView 选中同步，先采用稳定可见的“当前”描述 + 图标标记。
- **右键菜单**：为题目节点增加包装命令，由菜单把 task tree 节点对象传入包装命令，再转调既有 task 级命令。

## 3. 验收标准

1. 点击题目打开后，树中对应题目显示“当前”
2. 题目节点右键可见上述 4 个动作
3. 菜单动作可直接作用于对应 taskRoot
